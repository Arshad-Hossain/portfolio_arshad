## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/Arshad-Hossain/YOUR_REPOSITORY_NAME.git
```

Navigate into the project:

```bash
cd YOUR_REPOSITORY_NAME
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```
